# Simple POS Desktop
A simple POS (Point Of Sales) software writen in C++ with Qt Framework.

## Key Features
* Multi users
* Product and categories
* Suppliers with balance
* Customers with balance
* Purchasing order
* Sales order
* Database backup, restore, reset
* Database SQLite / MySQL
